# CLAUDE.md

## Tech Stack
- Angular (latest)
- NgRx Store (latest)
- Angular Material (latest)

## App Overview
- Mobile-first, frontend-only birthday minigame app
- Hosted on GitHub Pages — must be lightweight
- 3 levels (minigames), completing all 3 unlocks a reward (downloadable PDF voucher)
- Games: "Skąd to zdjęcie", "Jaka to melodia", "Rebus"

## Language
- All UI content in Polish
- All code (variables, functions, comments) in English

## Rules
- Use signal-based approach whenever possible (signal inputs, computed signals, effect, model signals)
- Lifecycle hooks are FORBIDDEN - use signals/effects instead
- Reports to user: extremely concise, sacrifice grammar for brevity
